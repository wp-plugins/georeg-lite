== Description ==

A plugin that transparently captures detailed geographic information when a new user registers.

GeoReg is built by the AppThemes team who also offers a full-blown [GeoReg version](http://marketplace.appthemes.com/plugins/georeg/ "GeoReg geographic info plugin from AppThemes") in their popular plugins and child-themes [Marketplace](http://marketplace.appthemes.com/ "WordPress Plugins and Child-Themes from AppThemes").
